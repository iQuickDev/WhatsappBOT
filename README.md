| Command      | Usage                                                       |
| ------------ | ----------------------------------------------------------- |

| fakeidentity | Generates a fake identity                                   |
| getrequest   | Send a GET request to a website                             |
| help         | Displays help                                               |
| igprofile    | Fetches profile information from IG                         |
